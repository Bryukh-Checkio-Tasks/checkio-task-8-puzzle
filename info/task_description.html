8 puzzle is a sliding puzzle that consists of a frame of randomly ordered, numbered square tiles with one missing tile. The object of the puzzle is to place the tiles in order (see picture) by using sliding moves to utilize the empty space. You can read more about this puzzle on Wikipedia

The puzzle is presented as a 3x3 matrix with numbers from 1 to 8. Zero is the empty cell. You can "move" the empty cell in four directions: up--"U", down--"D", left--"L", and right--"R." You need to return a sequence of moves to solve the puzzle. The solution is presented as string of symbols ("UDLR") describing your moves.

Input: A matrix with numbers from 1 to 8, a list of lists with integers.

Output: A string describing the route of the empty cell.

Example: 

checkio([[1, 2, 3],
         [4, 6, 8],
         [7, 5, 0]]) == "ULDR"
